# Quantum Resistance in Cryptographic Enclaves

This paper discusses the implementation and security bounds of post-quantum cryptography inside secure hardware boundaries (Secure Enclaves).

## Computational Metrics

:::violet
### Key Generation Overhead
Lattice-based algorithms present an increased computational overhead during key generation phases. We observe a 2.4x latency increase compared to standard Elliptic Curve Cryptography.
:::

Our scientific analysis demonstrates that post-quantum enclaves are highly viable for secure local data processing.
